# Installation

```
npm install mads-widgets
```

# Full Bundle

```
import Vue from 'vue'
import MadsWidgets from 'mads-widgets'

Vue.use(MadsWidgets)
```

# Individual components as Vue plugins

```
import Vue from 'vue'
import { MadsHighchart } from 'mads-widgets'

Vue.use(MadsHighchart)
```
